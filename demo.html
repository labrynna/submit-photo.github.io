How It Works

1

Upload Photo

User takes a photo showing developer information at a construction site. Mobile devices can use the camera directly, or users can upload an existing photo.

2

Analyze & Extract

Google Cloud Vision API extracts all text from the image. The application then intelligently parses this text to identify:

  • Company name (looks for LLC, Inc, Corp, etc.)
  • Website URLs (detects .com, .net, .org, etc.)
  • Phone numbers (various formats: (555) 123-4567, 555-123-4567, etc.)
  • Street addresses (number + street name patterns)
3

Match or Create

The application searches the Google Sheet for an existing entry matching the address. If found, it prepares to update that entry. If not found, it will create a new row.

4

Review & Edit

User reviews the automatically extracted information and can make corrections before saving. This ensures data accuracy.

5

Save to Sheet

Verified data is saved to the Google Sheet with columns: Address, Company Name, Website, Phone, Date Added, and Photo Text.
